## 1. Introduction

This project is a full-stack hospital management web application built using Python and Flask. It provides doctors and hospital staff with a digital platform to manage patient records, schedule appointments, record diagnoses, store lab reports, and detect malaria from blood smear images using an AI model trained with deep learning.

The system replaces paper-based hospital records with a secure, database-connected web interface. All data is stored in a MySQL database. Access is protected by a login system with bcrypt password hashing and an invite-code-based doctor registration system.

The highlight of the project is the AI Malaria Detector — a Convolutional Neural Network (CNN) trained on 27,558 blood cell images that achieves **97.6% accuracy**, integrated directly into the hospital web application.

## 5. MySQL Database Design

**Database name:** `electronic_health_records`

The database contains 9 tables. All clinical tables link back to the `patients` table using `patient_id` as a foreign key.

### Table Overview

| Table | Purpose |
|-------|---------|
| patients | Core patient demographics and contact info |
| users | Doctor/admin login accounts |
| invite_codes | One-time registration codes generated by admin |
| appointments | Scheduled appointments |
| diagnosis | Clinical diagnosis records |
| lab_reports | Blood test and imaging results |
| medical_history | Past medical history entries |
| treatment | Treatment plans and prescriptions |
| malaria_ai_results | AI malaria detection results with image path |

### Key Table: patients

```sql

    patient_firstname  VARCHAR(100) NOT NULL,

);
```

### Key Table: users

```sql
CREATE TABLE IF NOT EXISTS users (
    id            INT AUTO_INCREMENT PRIMARY KEY,
    username      VARCHAR(100) NOT NULL UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    full_name     VARCHAR(150),
    role          ENUM('admin', 'doctor', 'nurse') DEFAULT 'doctor',
    created_at    TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Key Table: malaria_ai_results

```sql
CREATE TABLE IF NOT EXISTS malaria_ai_results (
    id           INT AUTO_INCREMENT PRIMARY KEY,
    patient_id   VARCHAR(50),
    patient_name VARCHAR(200),
    age          INT,
    gender       VARCHAR(10),
    result       VARCHAR(20),
    confidence   DECIMAL(5,2),
    severity     VARCHAR(20),
    treatment    TEXT,
    image_path   VARCHAR(500),
    analyzed_by  VARCHAR(100),
    analysis_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (patient_id) REFERENCES patients(patient_id) ON DELETE SET NULL
);
```

```python
def verify_user(username, password):
```

### Auto-Generated IDs

Every patient gets a unique ID in the format `YYYYMMDDNN` (date + sequence number):

```python
def generate_patient_id(cursor):
    today = datetime.now().strftime('%Y%m%d')
    cursor.execute("SELECT COUNT(*) FROM patients WHERE patient_id LIKE %s", (f"{today}%",))
    count = cursor.fetchone()[0]
    return f"{today}{count + 1:02d}"
```

All other modules (appointments, diagnosis, lab reports, etc.) get IDs in the format `PREFIX-YYYYMMDD-NNN`:

```python
def generate_record_id(prefix, cursor, table, id_col):
    today = datetime.now().strftime('%Y%m%d')
    cursor.execute(f"SELECT COUNT(*) FROM {table} WHERE {id_col} LIKE %s", (f"{prefix}-{today}%",))
    count = cursor.fetchone()[0]
    return f"{prefix}-{today}-{count + 1:03d}"
```

Examples: `APT-20260316-001`, `DIA-20260316-002`, `LAB-20260316-001`

## 8. AI Malaria Detection — CNN Model

### 8.1 Dataset

The model was trained on the **NIH Malaria Cell Images Dataset**, which contains **27,558 blood smear cell images** — 13,779 Parasitized (infected) and 13,779 Uninfected images. The dataset was split 80% for training and 20% for validation, giving 22,046 training images and 5,512 validation images.

### 8.2 Data Preprocessing and Augmentation

Before training, images were preprocessed and augmented using `ImageDataGenerator`:

```python
datagen = ImageDataGenerator(
    rescale=1./255,          # normalize pixel values from 0-255 to 0-1
    validation_split=0.2,    # 20% for validation
    rotation_range=15,       # randomly rotate images up to 15 degrees
    zoom_range=0.1,          # randomly zoom in/out by 10%
    width_shift_range=0.05,  # slight horizontal shift
    height_shift_range=0.05, # slight vertical shift
    horizontal_flip=True     # randomly flip images horizontally
)
```

Augmentation artificially increases the variety of training data, helping the model generalize better to new images it has never seen before.

### 8.3 Model Architecture

A custom Convolutional Neural Network (CNN) was designed with 4 convolutional blocks followed by a classification head:

```python
model = Sequential([
    # Block 1 — detects basic edges and textures
    Conv2D(32, (3,3), padding='same', input_shape=(128,128,3)),
    BatchNormalization(),
    Activation('relu'),
    MaxPooling2D(2,2),

    # Block 2 — detects shapes and patterns
    Conv2D(64, (3,3), padding='same'),
    BatchNormalization(),
    Activation('relu'),
    MaxPooling2D(2,2),

    # Block 3 — detects complex cell features
    Conv2D(128, (3,3), padding='same'),
    BatchNormalization(),
    Activation('relu'),
    MaxPooling2D(2,2),

    # Block 4 — high-level feature extraction
    Conv2D(256, (3,3), padding='same'),
    BatchNormalization(),
    Activation('relu'),
    MaxPooling2D(2,2),

    # Classification head
    GlobalAveragePooling2D(),
    Dense(128),
    BatchNormalization(),
    Activation('relu'),
    Dropout(0.5),
    Dense(1, activation='sigmoid')   # output: probability of Uninfected
])
```

**Why each component was chosen:**

- `Conv2D` — scans the image with small filters to detect features like edges, shapes, and textures
- `BatchNormalization` — normalizes layer outputs to stabilize and speed up training
- `MaxPooling2D` — reduces image size by half at each step, keeping only the strongest features
- `GlobalAveragePooling2D` — replaces Flatten to reduce parameters and prevent overfitting
- `Dropout(0.5)` — randomly turns off 50% of neurons during training to prevent memorization
- `sigmoid` — outputs a value between 0 and 1 (probability), used for binary classification

### 8.4 Training Configuration

```python
model.compile(
    optimizer=Adam(learning_rate=1e-4),
    loss='binary_crossentropy',
    metrics=['accuracy']
)

class_weight = {0: 2.0, 1: 1.0}

early_stop = EarlyStopping(
    monitor='val_loss',
    patience=5,
    restore_best_weights=True
)

history = model.fit(
    train_generator,
    validation_data=val_generator,
    epochs=12,
    class_weight=class_weight,
    callbacks=[early_stop]
)
```

**Key training decisions:**

- `Adam optimizer` with `learning_rate=1e-4` — adaptive optimizer that adjusts learning rate automatically for each parameter
- `binary_crossentropy` loss — standard loss function for binary (two-class) classification
- `class_weight = {0: 2.0, 1: 1.0}` — penalizes the model twice as much for missing an infected case, because in medical diagnosis a false negative (missing malaria) is far more dangerous than a false positive
- `EarlyStopping` with `restore_best_weights=True` — stops training when validation loss stops improving and automatically restores the best model weights

## 9. Secure Login System

Passwords are never stored as plain text. bcrypt is used to hash passwords before storing them in the database.

```python
import bcrypt

# Creating a new account — hash the password
hashed = bcrypt.hashpw(password.encode('utf-8'), bcrypt.gensalt()).decode('utf-8')
cursor.execute("INSERT INTO users (username, password_hash) VALUES (%s, %s)", (username, hashed))

# Logging in — verify the entered password against the stored hash
bcrypt.checkpw(entered_password.encode('utf-8'), stored_hash.encode('utf-8'))
# Returns True if correct, False if wrong
```

Even if the database is compromised, the actual passwords cannot be recovered from bcrypt hashes.

## 10. Invite Code System for Doctor Registration

To prevent unauthorized account creation, doctors cannot register freely. Only the admin can generate invite codes.

**Admin generates a one-time code:**

```python
import secrets
from datetime import timedelta

code = secrets.token_hex(8).upper()          # e.g. "A3F9B2C1D4E5F6A7"
expires_at = datetime.now() + timedelta(hours=1)
cursor.execute(
    "INSERT INTO invite_codes (code, role, created_by, expires_at) VALUES (%s, %s, %s, %s)",
    (code, 'doctor', session['username'], expires_at)

```

**Doctor registers using the code:**

```python
cursor.execute("""
```

## 11. Live Patient Autocomplete Search

A live search API allows doctors to find registered patients by typing partial names. The patient ID is then auto-filled into the form.

**Flask API endpoint:**

```python
@app.route('/api/search-patients')
def api_search_patients():
    q = request.args.get('q', '').strip()
    if len(q) < 2:
        return jsonify([])
    like = f"%{q}%"
    cursor.execute("""
        SELECT patient_id,
               CONCAT(patient_firstname, ' ', patient_lastname) AS full_name,
               patient_DOB, phone
        FROM patients
        WHERE patient_firstname LIKE %s OR patient_lastname LIKE %s
        LIMIT 10
    """, (like, like))
    return jsonify(cursor.fetchall())
```

**JavaScript calls this API as the user types:**

```javascript
nameInput.addEventListener('input', () => {
    if (nameInput.value.length < 2) return;
    fetch(`/api/search-patients?q=${nameInput.value}`)
        .then(r => r.json())
        .then(data => {
            // show dropdown with matching patients
            data.forEach(p => {
                let item = document.createElement('div');
                item.textContent = `${p.full_name} — ${p.patient_id}`;
                item.onclick = () => {
                    nameInput.value = p.full_name;
                    document.getElementById('patient_id').value = p.patient_id;
                };
                dropdown.appendChild(item);
            });
        });
});
```

This autocomplete is active on all 5 clinical module forms: Appointment, Diagnosis, Lab Report, Medical History, Treatment, and the Malaria Detector.


Jinja2 syntax used:
